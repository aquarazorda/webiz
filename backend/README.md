
# Usage
 Install the PureScript Compiler
 ```
npm install -g purescript@0.14.1
```
Install npm packages
```
npm install
```
Install package manager for PureScript
```
npm install -g spago
```
Install spago packages
```
spago install
```
Run express server
```
spago build
spago run
```
Server will start on 8080 port
